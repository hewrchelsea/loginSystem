## Installation

- You need to import the 'database.sql' file into a mysql database with the same name in the database connection file in '/conn/conn.php';

- you need php mail function enabled, and add your own email change the email inside 'sendEmail/sendemail.php' to your email.

- change the password reset url to your url in "forgot-password.php/requestNewPassword.php" on line 35.
